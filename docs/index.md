# Off-The-Path Travel Recommendation System

A hybrid travel discovery application using BM25 keyword search and ModernBERT semantic embeddings.

Features:

- Scrapes and stores travel blogs automatically
- BM25 baseline retriever for fast lexical search
- ModernBERT vector-based semantic search
- FastAPI backend + Streamlit UI
- Fully containerized with Docker
- Cloud-hosted Postgres + S3 embedding store
